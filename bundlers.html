Module Bundlers

¿Que son los Module Bundlers? Un "Module Bundler" es una herramienta que toma partes de JavaScript y sus dependencias y las agrupa en un solo archivo, generalmente para usar en el navegador. Por lo general, comienza con un archivo de entrada y, a partir de ahí, agrupa todo el código necesario para ese archivo de entrada.
